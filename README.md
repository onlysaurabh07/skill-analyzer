# Skill Analyzer

## About
Skill Analyzer is a Python-based application that analyzes resumes, extracts skills, and identifies skill gaps to help users improve their profiles.

## Features
- Resume analysis
- Skill extraction
- Skill gap identification
- Skill recommendations

## Technologies Used
- Python
- Pandas
- Scikit-learn
- HTML
- CSS
- nodejs
- mongodb
